## Project Overview
This project presents an evaluation of Python as an open-source software. It covers its history, philosophy, role in Linux systems, and overall ecosystem. The project also includes practical implementation through Bash scripting and a comparison with proprietary software solutions.

---

## Script Descriptions

### 1. System Information Script
This script gathers key system details, including:
- Kernel version  
- Active user  
- System uptime  
- Current date and time  

---

### 2. Open Source Package Checker
- Checks if Python is installed on the system  
- Displays package-related information  
- Uses conditional logic and case statements  

---

### 3. Disk & Permission Analyzer
- Scans important directories  
- Shows directory size and permissions  
- Utilizes loops and Linux commands  

---

### 4. Log File Processor
- Reads a specified log file  
- Counts occurrences of a given keyword  
- Outputs a summary along with recent matches  

---

### 5. Manifesto Generator
- Accepts user input  
- Creates a custom open-source manifesto  
- Saves the result into a text file  

---

## Requirements
- Ubuntu Linux or WSL  
- Bash shell environment  
- Python 3 installed  
- Familiarity with basic Linux commands (e.g., `ls`, `grep`, `awk`)

## Execution Guide

### Step 1: Launch Terminal
Open your Ubuntu or WSL terminal.

### Step 2: Navigate to Project Directory
```bash
cd path/to/your/scripts
```
### Step 3: Grant Execution Permissions
```
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh
```
### Step 4: Execute Scripts
```
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog error
./script5.sh
```
